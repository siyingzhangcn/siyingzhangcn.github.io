Exhaustive search

  • Explore decisions (pseudo-code):

      - if there are no more decision to make: stop;
  - else; let's handle one decision ourselves and the rest by recursion.
      for each available choice C for this desicion:
          ~ <font color=red><Choose></font> C
          ~ <font color=red><Explore></font> C

  • Helper functions:

    • if requried function dose not accept the parameters you need:
      • write a helper function that accept more parameters
      • extra params can represent current state, choice made, etc.

Recursive Backtracking

  • Explore decisions (pseudo-code) :
      - if there are no more decision to make: stop;
  - else; let's handle one decision ourselves and the rest by recursion.
      for each available choice C for this desicion :
          ~ <font color=red><Choose></font> C
          ~ <font color=red><Explore></font> the remaining decisions that could follow C
          ~ <font color=red><Un-choose></font> C (backtracking!)
